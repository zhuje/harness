# Projects

- [console](console): The web console for the Openshift Container Platform, the main runtime of the dynamic plugins loaded using webpack module
  federation.
- [konflux-coo](konflux-coo): Productization pipeline for COO (cluster observability operator) running in konflux, the internal CI/CD system of Red
  Hat.
- [logging-view-plugin](logging-view-plugin): An Openshift Web console dynamic plugin to include logging capabilities in the console using the Loki
  stack.
- [monitoring-plugin](monitoring-plugin): An Openshift Web console dyanmic plugin to include metrics, alerts, silences, targets and dashboards (perses
  and legacy) in the console.
- [observability-operator](observability-operator):Also known as Cluster Observability Operator (COO). An upstream open-source k8s operator that
  manages observability resources, including monitoring stacks for Prometheus and Perses dashboards by embedding the Perses operator.
- [perses](perses): An open-source project for creating and managing dashboards and visualizations for monitoring.
- [perses-operator](perses-operator): An upstream open-source k8s operator that manages Perses resources, including perses backend instances,
  datasources, global datasource and dashboards.
- [perses-plugins](perses-plugins): A collection of plugin modules for Perses, including datasources, panels and variables.
- [perses-shared](perses-shared): A collection of shared libraries for Perses plugins, including plugin system, components, dashboards and utilities.
- [perses-specs](perses-specs): A collection of CUE specifications for Perses resources, including dashboards, datasources and plugins.
